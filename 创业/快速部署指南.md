# 🚀 快速部署指南 - 让外部访问你的网站

## ⚡ 最快方法：Netlify 拖拽部署（推荐）

### 步骤（3分钟完成）：

1. **打开 Netlify 拖拽页面**
   - 访问：https://app.netlify.com/drop
   - 无需注册，直接拖拽即可

2. **拖拽整个项目文件夹**
   - 将包含以下内容的整个文件夹拖进去：
     - ✅ index.html
     - ✅ styles.css
     - ✅ script.js
     - ✅ 图片文件夹（包含所有图片）

3. **等待部署**
   - 页面会显示 "Deploying..."
   - 大约 1-2 分钟后显示 "Published"

4. **获得链接**
   - 部署完成后会显示类似：`https://xxxxx.netlify.app`
   - 这个链接可以分享给任何人访问！

5. **（可选）自定义域名**
   - 点击 "Site settings" → "Change site name"
   - 可以改成更友好的名字，如：`nz-travel.netlify.app`

## 📦 方法二：GitHub Pages（永久免费）

### 步骤：

1. **创建 GitHub 账号**
   - 访问 https://github.com 注册

2. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名：`nz-travel-website`
   - 选择 **Public**（公开）
   - 点击 "Create repository"

3. **上传文件**
   - 在仓库页面点击 "uploading an existing file"
   - 拖拽上传所有文件：
     - index.html
     - styles.css
     - script.js
     - 图片文件夹
   - 点击 "Commit changes"

4. **启用 GitHub Pages**
   - 进入 Settings → Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"，文件夹选择 "/ (root)"
   - 点击 Save

5. **获取链接**
   - 几分钟后访问：`https://你的用户名.github.io/nz-travel-website/`

## 🔧 方法三：临时测试 - 内网穿透

如果想临时让别人访问本地网站：

### 使用 ngrok：

1. **安装 ngrok**
   ```powershell
   # 下载地址：https://ngrok.com/download
   # 或使用 chocolatey
   choco install ngrok
   ```

2. **启动本地服务器**
   ```powershell
   python -m http.server 8000
   ```
   如果没有 Python，可以用：
   ```powershell
   npx http-server -p 8000
   ```

3. **启动 ngrok**
   ```powershell
   ngrok http 8000
   ```

4. **分享链接**
   - ngrok 会显示一个公网链接（如：`https://xxxx.ngrok.io`）
   - 这个链接可以分享给别人
   - ⚠️ 注意：免费版链接每次重启都会变化
